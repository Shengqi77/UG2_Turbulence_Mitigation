# UG2+ Track2: ATMOSPHERIC TURBULENCE MITIGATION
This is the code used for 2023 CVPR UG2+ TRACK2: Atmospheric Turbulence Mitigation. We achieved the first place in both subtracks.
<img width="864" alt="image" src="https://github.com/xsqhust/Turbulence_Removal/blob/main/figs/zhengshu.png">

Track Link：https://cvpr2023.ug2challenge.org/track2.html

## Instructions
The instructions for running the code for each subtrack are provided in the readme file within the respective folder.

## Technical Report
Subtrack1 Paper Link: https://arxiv.org/pdf/2306.08963.pdf

Subtrack2 Paper Link: https://arxiv.org/pdf/2306.09379.pdf 

## Datasets
The Turbulence Dataset can be downloaded from https://cvpr2023.ug2challenge.org/dataset23_t2.html
